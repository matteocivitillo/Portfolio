
# Portfolio - Matteo Civitillo

Welcome to my personal portfolio website! I'm a **Computer Engineering & HCID Student** passionate about user interfaces and modern digital experiences.

## 🌐 Website Features

This website showcases:
- **Home** - A brief introduction and call-to-action
- **About** - Information about my background and interests
- **Projects** - A collection of my recent work and projects
- **Skills** - Technologies and tools I work with
- **Contact** - Ways to get in touch with me

## 🛠️ Tech Stack

- **Frontend Framework**: React with TypeScript
- **Router**: React Router v7
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **Internationalization**: i18next (English & Italian)
- **Animations**: Framer Motion
- **UI Components**: Radix UI
- **Icons**: Tabler Icons, Lucide React

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173/Portfolio/`

### Build for Production

```bash
npm run build
```
